Hi,

This is just to inform R users that NCAR's verification group has created a library of verification functions in R.  It is available on the R website at http://www.r-project.org.  From the main page, go to CRAN (left banner), pick a nearby mirror site, then select contributed.  Also available at this site it all the other information you need to know about R.

If you have R installed you can get the package by typing install.packages(verification) in Linux.  Window user can select the package from those listed on CRAN.

This is a first version of the package and it contains functions for a lot of routine verification graphics and statistics.  These include ROC plots, attribute diagrams, reliability plots and conditional quantile plots along with documentation and examples.  New features include a measurement error function developed by William (Matt) Briggs and David Ruppert and a spatial scale verification function developed  by Barbara Casati (soon to be updated.)

I hope to update the package early next year and I would like some suggestions and feedback as to what might be useful.  Some suggestions have included, verification of circular (directional) forecasts (Tilmann Gneiting) and confidence intervals for everything. This last item will be the subject of another message.  If you work in R and have a routine or function that you would like to share, I would be happy to include it and help you with documentation and example creation.
